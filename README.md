# Cadastro de Curriculos

## Rodar

Instalar dependencias:

```bash
npm install
```

Criar um `.env` na raiz do projeto:

```env
VITE_SUPABASE_URL=https://SEU-PROJETO.supabase.co
VITE_SUPABASE_ANON_KEY=SUA_CHAVE_PUBLICA_ANON
```

Iniciar:

```bash
npm run dev
```

## Banco

SQL:

```text
supabase/schema.sql
```
